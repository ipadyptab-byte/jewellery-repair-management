RML Connect — WhatsApp send (WBA)
This is the API that actually sends template messages to customers. Use your messages URL and the API key from RML Connect (often a long JWT). What you type here is saved automatically in this browser (and when you click Save). Nothing clears unless you change it yourself. If you still see Invalid token after a new key, Route Mobile is rejecting that key — ask them for a send-enabled API key.
Must match what Route Mobile gave you for sending.
WhatsApp Business Account ID (if your integration requires it).
If you still see Invalid token, your browser may be using an old saved key — click Reset saved WBA to file defaults, then Save again. The JWT must be a send-enabled key from RML Connect.
Invoice PDF link base URL
The base URL where invoice PDFs are hosted. The app generates links like: https://invoice.devijewellers.in/INV-JR1001-abc123?expires=10d — customers click this to view their invoice. Use your hosting or cloud storage URL.
After this many days the invoice link will expire. Default: 10 days.
WhatsApp Business API requires pre-approved HSM templates. The invoice PDF link is automatically embedded as variable {{6}} in the received template and {{4}} in the ready template.
